# Vullam Teja - Personal Portfolio

A sleek, responsive, and minimalist personal portfolio website built to showcase my projects, technical skills, and background as a Computer Science Engineering undergraduate at Amrita Vishwa Vidyapeetham. The design follows premium, modern aesthetic principles inspired by an Apple/Stripe-like UI.

## Features

- Single-Page Application: A standalone index.html file containing all structure, styling, and logic.
- Responsive Design: Fully optimized for desktop, tablet, and mobile viewing.
- Dark Mode Toggle: Built-in dark and light theme switcher that detects system preference.
- Premium UI/UX: Features frosted-glass sticky navigation, smooth scrolling, and scroll-reveal animations using IntersectionObserver.
- Organized Sections: Dedicated areas for About, Technical Skills, Featured Projects, Education, and Contact Information.

## Technical Details

- Architecture: Pure HTML5, CSS3, and Vanilla JavaScript.
- External Dependencies: Zero framework dependencies. Only uses Google Fonts (Inter).
- Animations: Custom CSS keyframe morphing animations and JavaScript-based scroll reveal logic. 

## Featured Projects Included

1. Hostel Complaint Portal: Secure portal integrating OTP verifications, JWT authentication, AES encryption, and Role-Based Access Control.
2. Coin-Pilot: Interactive, data-driven platform for tracking cryptocurrency trends and analyzing portfolios.
3. Campus-Connect: Comprehensive university platform for communication, event updates, and resource sharing.

## Deployment

Because this project is a standalone HTML file without external framework dependencies, it can be hosted seamlessly on static hosting platforms such as:
- GitHub Pages
- Vercel
- Netlify

To deploy using GitHub Pages:
1. Navigate to the repository Settings.
2. Select Pages from the left sidebar.
3. Under Build and deployment, set the source to the main branch.
4. Save the changes. The site will be built and deployed automatically.
